專案名稱：公開資訊觀測站 (MOPS) 企業報告自動化抓取系統
專案簡介
本專案為一套基於 Python Selenium 開發的自動化 RPA 工具。主要解決手動從「公開資訊觀測站」抓取企業財務報告、法說會簡報及股東會年報時，面臨的介面繁瑣、多層視窗跳轉以及新舊版網頁架構不一的問題。系統能根據輸入清單，自動識別對應路徑並完成下載作業。

核心功能
多路徑自動識別：一套程式兼容「舊版選單 (iframe 架構)」與「新版 Vue 介面」三種不同的查詢流程。

複雜視窗控制：自動處理查詢彈窗、分頁跳轉（如 doc.twse.com.tw）以及最終 PDF 下載頁的標籤切換。

穩定性機制：

具備「環境強制清理」，確保每筆任務開始前瀏覽器狀態純淨。

內建防封鎖隨機延遲（Random Sleep），模擬真人操作行為。

自動偵測並排除清單中的重複檔案連結，避免冗餘下載。

異常處理：單一公司或檔案下載失敗時，系統會自動截獲錯誤並跳轉下一筆，確保整批任務不中斷。

技術棧
語言：Python 3.14+

自動化框架：Selenium WebDriver

環境管理：uv

資料處理：Pandas (Excel 讀取與篩選)

驅動管理：WebDriverManager (自動同步 Chrome 版本)

檔案結構說明
Main_Crawler.py: 核心執行程式。

trigger file.xlsx: 任務啟動表，包含「公司代號」、「年度」、「URL 識別碼」及「目標檔案關鍵字」。

/pdf: 預設檔案下載存儲目錄。

執行準備
確保已安裝 Python 3.14 與 uv。

使用 uv pip install selenium pandas webdriver-manager openpyxl 安裝依賴。

確認 Excel 中的 URL 欄位包含正確關鍵字（如 t100sb02_1 代表法說會）。
